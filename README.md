#Hotel Booking System

##In this project "Hotel Booking System" I have tried to show how the data in hotel is managed. 
It manages and maintains the records of customers, rooms, employees and drivers in the hotel.
Main Objective of this project is to provide solution for hotel to manage most of there work using
computerized process.

#As a security I have provided Admin username and password.
##The application of the Hotel Booking System bears the following functions to use by the Administrator :
 1.Add a new Room
 2.Add an Employee
 3.Add a new Customer
 4.Check room status
 5.Check all employees details
 6.Update room status
 7.Update check status etc.

##Technologies used:
-Java(AWT And Swing)
-MySQL
